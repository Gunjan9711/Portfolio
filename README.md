# 💼 Personal Portfolio Website

A modern, responsive, and animated personal portfolio website built using **HTML**, **CSS**, and **JavaScript**. The project showcases a clean UI with smooth scrolling, interactive animations, project showcases, testimonials, and a contact section.

## ✨ Features

- 🎨 Modern and responsive UI
- 📱 Mobile-friendly navigation menu
- ⚡ Smooth scrolling between sections
- 🌟 Scroll reveal animations
- 📌 Active navigation highlighting
- 🖼️ Hero section with parallax image effect
- 💼 Recent Work showcase
- 📖 Case Studies section
- 💬 Client Testimonials
- 📩 Contact form with basic validation
- 🎯 Interactive hover effects
- 📜 Custom scrollbar styling

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Google Fonts (Poppins)

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/your-username/portfolio.git
```

2. Open the project folder.

3. Open `index.html` in your browser.

No installation or dependencies are required.
